flex布局：

一行中，某一边固定宽度，另一边占当前行的剩余宽度。

```html
<div class="couponBox">
    <div class="cpLeft">
        11
    </div>
    <div class="cpRight">
        22
    </div>
</div>
```



```css
.couponBox {
    display: flex;
    background: #fff2f9;
}
.cpLeft {
    height: 76px;
    width: 108px; 
    /* 固定宽度 */
}
.cpRight {
    height: 76px;
    flex-grow: 1;
}
```
